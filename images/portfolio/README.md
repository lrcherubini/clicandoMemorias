# Portfolio Images

Esta pasta contém as galerias de imagens do portfólio organizadas por categoria.

## Estrutura

```
portfolio/
├── familias/          # Galeria de ensaios de família
├── gestantes/         # Galeria de ensaios gestantes
└── 15anos/            # Galeria de ensaios de 15 anos
```

## Como Adicionar Imagens

1. Coloque as fotos na pasta da categoria correspondente
2. Configure o arquivo `gallery.json` em cada pasta (veja exemplos nas pastas)
3. As imagens devem estar em formato JPG, PNG ou WEBP
4. Recomendado: otimize as imagens antes de fazer upload (max 2MB por imagem)

## Formatos Suportados

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Dicas

- Use nomes descritivos para os arquivos (ex: `familia-parque-1.jpg`)
- Ou use o padrão `foto-1.jpg`, `foto-2.jpg`, etc.
- Mantenha a qualidade alta, mas otimize o tamanho dos arquivos
- Adicione legendas significativas no `gallery.json`

Para mais informações, consulte o arquivo `GALLERY_GUIDE.md` na raiz do projeto.
